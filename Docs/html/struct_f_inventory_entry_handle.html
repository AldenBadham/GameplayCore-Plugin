Detailed Description

Represents a handle to an inventory entry, providing access to its index, item instance, and stack count.

See also
UItemInstance

This structure acts as a lightweight wrapper around inventory entry data, allowing access to the entry's position in the inventory, its associated item instance, and the current stack count. Used throughout the inventory system to reference and manipulate inventory entries.
