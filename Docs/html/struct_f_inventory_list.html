Detailed Description

A replicated container that manages a list of inventory entries.

See also
FFastArraySerializer

Handles the storage, replication, and management of inventory entries including addition, removal, and modification of items. Utilizes FastArraySerializer for efficient network replication and provides interfaces for item manipulation and querying.
