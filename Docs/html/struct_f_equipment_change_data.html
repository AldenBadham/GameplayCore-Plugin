Detailed Description

Represents a data payload for tracking Equipment item changes including addition, removal, and modification of items.

See also
UItemInstance, FEquipmentEntry, EEquipmentChangeType

Contains information about the affected item instance, its index in the Equipment, the type of change, and stack count changes before and after the modification occurred. Used for notifying systems about Equipment state changes and maintaining synchronization.
