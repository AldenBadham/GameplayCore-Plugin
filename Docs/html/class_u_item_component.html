Detailed Description

Base class for modular components that can be added to item instances.

See also
UItemInstance

This class serves as the foundation for creating specialized components that add specific functionalities to item instances. Components are attached to item instances to extend their capabilities. Each component can implement its own replication logic and functionality while maintaining a modular design approach.
